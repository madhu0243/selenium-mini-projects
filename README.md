# Selenium Mini Projects 🚀

This repository contains beginner-friendly browser automation projects created using Python and Selenium.

## 📌 About

These projects demonstrate how to automate web browser tasks such as searching websites, interacting with elements, and capturing screenshots.
Automation is performed using Selenium WebDriver with Google Chrome.

## 🛠 Technologies Used

* Python
* Selenium
* WebDriver Manager
* Google Chrome

## 📂 Projects Included

### 1️⃣ Google Search Bot

Automatically opens Google and performs a search query.

File:

google_search_bot.py

Features:
* Opens Google
* Types search query automatically
* Displays search results

### 2️⃣ YouTube Search Bot
Automatically searches videos on YouTube.

File:

youtube_bot.py

Features:
* Opens YouTube
* Searches for a video
* Shows search results

### 3️⃣ Screenshot Bot
Opens a website and captures a screenshot.
File:

screenshot_bot.py

Features:
* Opens a webpage
* Takes screenshot automatically
* Saves image locally

## ⚙️ Installation
Install required Python packages:

pip install selenium
pip install webdriver-manager

## ▶️ How to Run
Run any project using Command Prompt:
python google_search_bot.py
or
python youtube_bot.py
or
python screenshot_bot.py
